Instructions to run the project

To run the project, you should have Unity 2019.9.11 and follow the steps below -

  1. Clone the repository from the above Github link
  2. Load the repository into unity as a project
  3. Open the project in unity
  4. Open the scene located at Assets/Scenes/StarsExplorer
  5. Run the project

You should see the scene in action. For now, to maintain the frame rates, I have only loaded the first 10k rows of the star data (because stars < 25 light years were far too less).
